# No Fly Zones

De volgende gebieden zijn aangemerkt als No Fly Zones.

1. Militare Basis
2. Gevangenis
3. Blokkenpark
4. De pier
5. Windmolenpark ( vanaf 400 meter geld de No Fly Zone)
6. Tweedehands
